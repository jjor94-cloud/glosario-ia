Glosario

¿Qué es un agente?

IA automática con un objetivo particular ejecutando tareas dentro de un entorno determinado.

¿Qué es un multiagente?

Sistema compuesto por múltiples agentes con objetivos y ejecuciones diferentes.

¿Para qué sirve un multiagente?

Permite delegar diferentes tareas en ejecuciones concretas y disciplinas variadas.

¿Qué es Droid?

Asistente autónomo que permite la ejecución de tareas complejas utilizando IA.

¿Qué es un Fake Bot?

Bot automatizado que simula el comportamiento de un humano.

¿Para qué sirve crear un repositorio?

Permite almacenar y gestionar cambios de proyectos en la nube.

¿Qué es GitHub?

Plataforma ampliamente utilizada para alojar repositorios de software.

¿Cómo funciona GitHub?

Permite almacenar código, gestionar cambios e integrar diferentes ramas de desarrollo.

¿Cuál es la diferencia entre Claude y Droid?

Claude es una IA generativa conversacional; Droid está orientado a la creación de agentes.

¿Qué es Open Cloud?

Plataforma que permite la generación y gestión de código abierto en la nube.

¿Qué es una sesión de uso en Ollama?

Sesión abierta que almacena el contexto y la conversación de una interacción.

¿Qué es un sistema predictivo de agentes?

Agente capaz de anticipar acciones relacionadas con el contexto de una conversación.

¿Para qué sirve hacer capacitaciones con un agente?

Permite convertir al agente en un experto enfocado en un objetivo específico.

¿Qué es un Skill?

Capacidad de ejecutar instrucciones concretas y especializadas dentro de una IA.
