¿Qué es un agente?

IA automática con un objetivo particular ejecutando tareas dentro de un entorno determinado.

¿Qué es un multiagente?

Sistema compuesto por múltiples agentes con diferentes objetivos y ejecuciones.

¿Para qué sirve un multiagente?

Permite delegar tareas especializadas en diferentes agentes.

¿Qué es Droid?

Asistente autónomo que ejecuta tareas complejas utilizando IA.

¿Qué es un Skill?

Capacidad específica que permite ejecutar instrucciones concretas dentro de un sistema de IA.
