Buscar Concepto

¿Qué es un agente?

IA automática con un objetivo particular ejecutando tareas dentro de un entorno determinado.

¿Qué es un multiagente?

Sistema compuesto por múltiples agentes con diferentes objetivos y ejecuciones.

¿Para qué sirve un multiagente?

Permite delegar tareas especializadas entre agentes.

¿Qué es Droid?

Asistente autónomo para la ejecución de tareas complejas mediante IA.

¿Qué es un Fake Bot?

Bot automatizado que simula el comportamiento humano.
