Glosario Completo

¿Qué es un agente?

IA automática con un objetivo particular ejecutando tareas de entorno.

¿Qué es un multiagente?

Sistema de múltiples agentes con diferentes objetivos y ejecuciones diferentes.

¿Para qué sirve un multiagente?

Para delegar diferentes tareas en ejecuciones concretas y disciplinas varias.

¿Qué es Droid?

Asistente autónomo que permite ejecución de tareas complejas con IA.

¿Qué es un Fake Bot?

Bot automatizado que se hace pasar por humano.

¿Para qué sirve crear un repositorio?

Biblioteca en la nube que permite almacenar cambios.

¿Qué es GitHub?

Repositorio muy utilizado para contener cambios de software en la nube.

¿Cómo funciona GitHub?

Almacena código de software y permite integrar diferentes ramas.

¿Cuál es la diferencia entre Claude y Droid?

Claude es generativa y conversacional; Droid es generadora de agentes.

¿Qué es Open Cloud?

Plataforma que permite la generación de código abierto.

¿Qué es una sesión de uso en Ollama?

Sesión abierta que permite almacenar la conversación puntual.

¿Qué es un sistema predictivo de agentes?

Agente que predice acciones relacionadas dentro de una conversación.

¿Para qué sirve hacer capacitaciones con un agente?

Permite reducir contexto en sesión y convertirlo en un experto orientado a objetivos.

¿Qué es un Skill?

Capacidad de ejecutar instrucciones concretas dentro de una IA.
